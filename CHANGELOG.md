# Changelog

## v0.2 - 2026-06-25

### 新内容

- 新增“银行贷款”：可以根据机队和航线规模申请贷款，扩张更快，但每季度需要支付利息。
- 新增“辣豆红包”：一次性资金援助，适合开局或现金紧张时救急。
- 新增“分部/基地”玩法：除了总部，还可以在其他城市开设分部；航线只能从总部或分部起飞。
- 新增“飞机租赁”：可以用更低的前期成本扩充机队，但租赁数量受自有飞机数量限制。
- 新增“飞机交付”：购买飞机后不再立即入队，需要等待交付；租赁飞机仍可立即使用。
- 新增机型服役和退役年份：不同年代可购买的飞机会随时间变化。
- 新增航空新闻：新机型服役、经典机型退役会出现在报纸中。
- 新增季度总结：推进回合后会同时看到新季度报纸和上一季度财报。
- 新增财报/报纸重读：可以随时回看当季报纸和上次财报；读档后也能继续重读上次财报。
- 新增飞机交付通知：飞机完成交付后会有提醒。

### 体验优化

- 游戏名称和默认公司名更新为“空豆霸业 / 豆豆航空”。
- 城市和飞机数据同步 upstream 的新版本。
- 购买飞机界面按厂商分组，并默认展开当前资金可负担的机型组。
- 顶部 HUD 优化显示，避免在窄窗口下文字被挤成逐字换行。
- 机队数量显示统一：HUD、机队面板和财报现在口径一致。
- 开局选择总部时，右侧城市列表现在可以直接点击选择城市。
- 地图保留当前工程版的矢量底图、缩放、拖动、横向循环和城市热区体验。
- 移动端地图支持单指拖动、双指捏合缩放，并优化相近城市的触控选择。
- 优化移动端地图拖动和缩放时的流畅度。
- 移动端底部操作收敛为“下一季”，减少重复入口，主要经营操作回到面板中完成。
- 移动端帮助按钮固定在底部状态栏右侧，操作提示改为单独一行显示。
- 面板滚动条更醒目，移动端浏览下方面板时更容易发现可滚动内容。
- 新手引导支持“知道了 / 不再提示”，并提供“？”帮助入口，可随时查看玩法说明或重新开启提示。
- 新手引导提示更完整，会提示开航线、买飞机、推进季度、开设分部等关键操作。
- 财报会显示贷款利息、贷款余额、购买/租赁机队数量等信息。
- 关闭分部前会提示受影响航线和飞机。

### 规则调整

- 航线只能从总部或分部起飞。
- 关闭分部会关闭从该分部起飞的航线，相关飞机回到空闲状态。
- 租赁飞机不能当作自有飞机出售，只能退租。
- 贷款额度与航线和机队规模相关。
- 贷款利息会进入季度成本。
- 飞机购买会受当前年份和服役期限制。
- 票价必须是有效的正整数。

### Bug 修复

- 修复购买飞机后 HUD 机队数量和机队面板/财报不一致的问题。
- 修复重读财报时可能混用旧利润和当前现金/机队状态的问题。
- 修复读档后无法可靠重读上次财报的问题。
- 修复开局选择总部时右侧城市列表无法点击的问题。
- 修复还款金额为 0 时仍可能显示成功的问题。
- 修复租赁飞机可能被错误出售的风险。
- 修复旧存档里非法或重复分部影响分部数量和费用的问题。
- 修复旧存档里缺少飞机分配字段时可能导致可用飞机计算异常的问题。
- 修复跨季度总结中报纸和财报时间口径不清的问题。
- 修复移动端相近城市触控热区重叠时可能选错城市的问题。
- 修复选择总部或分部时新手引导可能遮挡操作的问题。

## v0.1 - 2026-06-24

### 首个本地版本

- 基于 upstream 原始版本整理出本地可运行版本。
- 普通玩家可以直接双击 `dist/standalone.html` 开始游戏。
- 保留原版航空经营核心玩法：选择时代、选择总部、购买飞机、开通航线、调整票价、推进季度。
- 加入本地存档和读档。
- 加入季度报纸事件和财报。
- 加入 AI 竞争航线。
- 加入深色管理面板风格界面。
- 加入世界地图、城市点、航线显示和城市选择。

### 地图更新

- 地图改为本地矢量底图，不再依赖运行时下载。
- 支持城市点击、hover 城市名、滚轮缩放、拖动和横向循环滚动。
- 改善手机触控热区，城市点更容易点击。

### 稳定性

- 补充了经济、航线、事件、存档和回合推进等基础测试。
- 构建流程可以生成普通静态版本和单文件 standalone 版本。
