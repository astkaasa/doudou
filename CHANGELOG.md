# Changelog

## v1.4 - 2026-07-10

### 新内容

- 四个时代剧本现在会在标示的结束年份进行时代航程结算，汇总主线阶段、公司市值、航线、机队和基地覆盖。
- 时代结算后可选择正式收官，或解除时代限制进入沙箱模式继续经营；选择结果会自动保存。

### 规则调整

- 新手引导改为直接使用开局自带飞机开通首条航线，不再要求先进行一次非必要购机。
- 首条航线开通后若公司季度预估仍亏损且有空闲飞机，引导会建议补充航线或调整票价。
- 首回合没有运营航线时增加风险确认，避免误操作导致空耗固定成本。
- 最终季度财报确认后必须处理一次时代结算，结算完成前不会继续推进季度。
- 辣豆基金改为每季度固定 0.5M 加航线营业收入的 2.5%，不再随现金余额复利增长。
- 航线客舱服务成本改为随实际旅客数和航程增长，大型机与长途航线不再只承担近似固定的餐食成本。
- “苍穹之路”的阶段目标与通关评级按四个时代分别校准，短剧本不再被过高的统一目标卡住，传奇剧本也不再沿用短剧本评级时限。
- “苍穹之路”的覆盖目标改为统计实际航线网络，不再强制以昂贵分部数量代表跨洲经营，并按五个区域总部的正式模拟结果重新校准规模目标。

### 体验优化

- 窄屏地图会在开局后自动定位总部，并可随时通过底部定位按钮返回总部。
- 飞机市场新增交付时效、真实租赁首期成本、购后现金说明，并禁用当前不可执行的交易。
- 航线开通界面新增客座率、季度利润和开通后现金四格预估，底栏新增季度经营预估。
- 弹层打开后会锁定背景交互、接管焦点，并支持使用 Esc 关闭可取消弹层。
- 最终季度财报增加“查看时代结算”入口，结算界面集中展示本局经营结果。
- 优化航线键生成与 AI 竞争航线查询，缩短大规模航网下的季度推进等待时间。
- 扩大桌面侧栏宽度并优化飞机市场在手机上的信息布局。

### Bug 修复

- 季度预估不再混入上一季度的随机故障损失和一次性运营支出。
- 存档现在会按版本顺序迁移并保存随机序列进度，覆盖存档前保留上一份备份；主存档损坏时可自动恢复备份，较新版本存档会被安全拒绝。
- 已超过时代期限的旧存档会迁移为沙箱续玩状态，不会在读档时被追溯结算打断。

## v1.3 - 2026-07-09

### 新内容

- 新增投资管理系统，可在城市新设、收购子公司，或在总部/分部城市投资机场建设。
- 新增公司市值统计，净资产由现金、自有飞机、子公司、证券持仓和贷款共同决定。
- 新增渐进式破产清算：先尝试急救贷款，再依次变卖证券、子公司和自有飞机，最后才触发天使投资或游戏结束。

### 规则调整

- 旅行社会提升同城航线客座率，机场建设会降低同城航线着陆费，免税店会带来季度品牌成长。
- 子公司按季度结算估值变化、回报收入和维护支出，新建子公司首季不产生回报。
- 苍穹之路的财富维度改用公司市值，不再只看现金余额。

### 体验优化

- 右侧快捷操作新增投资管理入口，地图会用金点标出已持有子公司的城市。
- HUD 新增市值徽章，财报新增子公司分区和公司市值速览。
- 帮助面板和机制发现卡新增子公司与公司市值说明。

## v1.1 - 2026-07-03

### 新内容

- 新增运营管理系统：员工人数、士气、服务预算、维修预算和广告预算会共同影响运营效能。
- 新增年度合同卡片：年度招聘和年终奖金从弹窗改为右下角合同卡片，待签署时会阻止推进季度。
- 新增飞机故障和事故后效，维修预算与运营效能会影响故障风险。

### 规则调整

- 航线维护和机组固定成本迁入运营预算，航线单项成本只保留燃油、起降和餐食。
- 落地费参数下调，早期航线利润率更接近上游新平衡。
- 扩张机队、航线和分部时会自动补充部分员工，避免首回合运营效能触底。

### 体验优化

- HUD 新增运营效能徽章。
- 右侧快捷操作新增运营管理入口，可调整三档运营预算。
- 财报新增运营预算、故障损失和运营效能信息。

### Bug 修复

- 旧存档会自动补齐运营管理字段，并迁移旧的待处理运营弹窗状态。
- 年度招聘与年终奖金合同按推进后的 Q3/Q4 触发，避免招聘错过旺季、奖金延到开年。
- 读档后会重新浮现待签署的年度合同卡片。

## v1.0 - 2026-07-02

### 新内容

- 新增主线任务系统“苍穹之路”：三阶段目标从“新手启航”推进到“苍穹之巅”。
- 主线进度同时考察资金、航线数、分部覆盖和持续盈利，四个维度全部达标后自动晋级。
- 第三阶段达成后触发通关评级，根据经营季度给出 S/A/B/C 评级和称号，并可选择收官或继续经营。

### 规则调整

- 新增洲际航线收益系数：跨区域长途航线获得 1.15x 到 1.50x 收益加成。
- 同区域跨经济圈且距离超过 3000km 的长途航线获得 1.12x 收益加成。

### 体验优化

- HUD 新增“苍穹之路”入口，可查看四维度目标、达标状态和阶段进度。
- 证券市场买入和卖出快捷数量新增 100M 档位。
- 财报基地收益的总部和分部标记改为更直观的图标。

### Bug 修复

- 旧存档会自动补齐主线任务字段，避免读档后缺失主线进度。
- 修正盛事预告里夏奥事件标题的名称显示。

## v0.9 - 2026-07-01

### 新内容

- 新增盛事事件系统：夏季奥运会和综合类世博会会按历史季度触发主办城市需求热潮。
- 盛事需求会按预热、开幕、落幕和余温阶段渐进变化，并影响股票板块。
- 盛事会对同区域城市和跨区域枢纽航线产生较小溢出加成。
- 新增城市：大阪、塞维利亚、汉诺威、米兰、蒙特利尔、里约热内卢。

### 规则调整

- 盛事开幕季度的主办城市所在细分区域不会被随机灾害新闻命中。

### 体验优化

- HUD 新增盛事徽章，多个盛事同时活跃时会轮播显示。
- 地图会用金色脉冲环标记盛事主办城市。
- 报纸会优先展示盛事头条，并使用专门的盛事版式。

### Bug 修复

- 旧存档会自动补齐新城市市场数据，并重建当前季度应生效的盛事修正器。

## v0.4 - 2026-06-28

### 新内容

- 新增“里程碑”奖杯陈列室，会记录航线、经济、机队、分部、财富和生涯目标。
- 新增随时代变化并按季度增长的城市市场数据，航线需求会跟随人口、商务和旅游热度变化。
- 同步 upstream 城市数据，新增河内、卡拉奇、阿姆斯特丹、旧金山、拉各斯等城市。
- 灾害新闻扩展到东亚、东南亚、南亚、中东、加勒比等细分区域。

### 规则调整

- 开设分部现在需要 1 个季度施工，完工后才可作为航线起飞基地。
- 航线收入改为按距离分层的班次模型，短途高频、远程低频；票价建议、开航成本、航线落地费和维护成本同步 upstream 最新参数。
- 各时代初始现金同步 upstream 最新参数，2000 年代开局资金更充裕。
- “辣豆基金”改为按当前现金 10% 固定结算，不再随机给出 1-100M。
- 中东、南亚等城市归入亚洲下的细分区域，灾害和需求事件按细分区域匹配。
- 灾害事件现在通过事件修正器让受影响区域航线临时停飞。
- 银行贷款弹窗不再提供一次性红包资金入口。

### 体验优化

- HUD 和重读按钮改为“上季利润 / 上季报纸 / 上季财报”，季度结算口径更清楚。
- 财报航线明细改为按总部/分部汇总基地收益，更容易看出各基地表现。
- 基地收益现在可以展开查看单条航线明细，便于定位赚钱或亏损航线。
- 城市和航线选择增加市场热度、趋势以及人口/商务/旅游提示，开线前更容易判断市场潜力。
- 移动端 HUD 压缩里程碑、存档和读档等低优先级文字，保留核心经营指标可读。
- 城市选择列表优先按细分区域分组，新增城市后更容易查找。

### Bug 修复

- 修正“辣豆豆”特质说明，使其与当前资金 10% 的辣豆基金规则一致。
- 修正负现金时“辣豆基金”可能变成额外扣款的问题。
- 修正航线频率收入计算，避免航线自身 frequency 与距离班次重复乘算。
- 修复移动端奖杯陈列室可能横向溢出的问题。

## v0.3 - 2026-06-27

### 新内容

- 新增城市：檀香山。
- 新增开局“金豆特质”：随机获得辣豆基金、维护费降低或油耗降低等公司特性。
- 新增航线管理表格：支持排序、分页、单独调价、停飞/复飞、换飞机和关闭确认。

### 体验优化

- 顶部 HUD 增加公司名和特质徽章，经营状态更明确。
- 金豆特质抽取会固定本局待选顺序，避免重复打开时重新随机。
- 移动端航线管理改为卡片式列表，调价、换机、停飞和关闭操作更适合触控。

- 开通航线现在需要支付一次性开通费用，长距离和高等级城市航线费用更高。

- 调价和更换执飞机型会保留上季度指标，航线管理中会标记变化，方便观察下一季度表现。

### Bug 修复

- 修复旧存档航线字段不完整时可能导致航线管理显示异常的问题。

## v0.2 - 2026-06-25

### 新内容

- 新增“银行贷款”：可以根据机队和航线规模申请贷款，扩张更快，但每季度需要支付利息。
- 新增“辣豆红包”：一次性资金援助，适合开局或现金紧张时救急。
- 新增“分部/基地”玩法：除了总部，还可以在其他城市开设分部；航线只能从总部或分部起飞。
- 新增“飞机租赁”：可以用更低的前期成本扩充机队，但租赁数量受自有飞机数量限制。
- 新增“飞机交付”：购买飞机后不再立即入队，需要等待交付；租赁飞机仍可立即使用。
- 新增机型服役和退役年份：不同年代可购买的飞机会随时间变化。
- 新增航空新闻：新机型服役、经典机型退役会出现在报纸中。
- 新增季度总结：推进回合后会同时看到新季度报纸和上一季度财报。
- 新增财报/报纸重读：可以随时回看当季报纸和上次财报；读档后也能继续重读上次财报。
- 新增飞机交付通知：飞机完成交付后会有提醒。
- 新增设置界面：可以切换地图样式和是否显示国界。

### 体验优化

- 游戏名称和默认公司名更新为“空豆霸业 / 豆豆航空”。
- 城市和飞机数据同步 upstream 的新版本。
- 购买飞机界面按厂商分组，并默认展开当前资金可负担的机型组。
- 顶部 HUD 优化显示，避免在窄窗口下文字被挤成逐字换行。
- 机队数量显示统一：HUD、机队面板和财报现在口径一致。
- 开局选择总部时，右侧城市列表现在可以直接点击选择城市。
- 地图保留当前工程版的矢量底图、缩放、拖动、横向循环和城市热区体验。
- 移动端地图支持单指拖动、双指捏合缩放，并优化相近城市的触控选择。
- 优化移动端地图拖动和缩放时的流畅度。
- 移动端底部操作收敛为“推进回合”，减少重复入口，主要经营操作回到面板中完成。
- 移动端推进按钮复用主界面的“推进回合 ▶”，保持文案和样式一致。
- 移动端帮助按钮固定在底部状态栏右侧，操作提示改为单独一行显示。
- 修正选择总部或分部时帮助按钮的位置，保持固定在底部状态栏最右侧。
- 面板滚动条更醒目，移动端浏览下方面板时更容易发现可滚动内容。
- 新手引导支持“知道了 / 不再提示”，并提供“？”帮助入口，可随时查看玩法说明或重新开启提示。
- 新手引导提示更完整，会提示开航线、买飞机、推进季度、开设分部等关键操作。
- 财报会显示贷款利息、贷款余额、购买/租赁机队数量等信息。
- 关闭分部前会提示受影响航线和飞机。

### 规则调整

- 航线只能从总部或分部起飞。
- 关闭分部会关闭从该分部起飞的航线，相关飞机回到空闲状态。
- 租赁飞机不能当作自有飞机出售，只能退租。
- 贷款额度与航线和机队规模相关。
- 贷款利息会进入季度成本。
- 飞机购买会受当前年份和服役期限制。
- 票价必须是有效的正整数。

### Bug 修复

- 修复购买飞机后 HUD 机队数量和机队面板/财报不一致的问题。
- 修复重读财报时可能混用旧利润和当前现金/机队状态的问题。
- 修复读档后无法可靠重读上次财报的问题。
- 修复开局选择总部时右侧城市列表无法点击的问题。
- 修复还款金额为 0 时仍可能显示成功的问题。
- 修复租赁飞机可能被错误出售的风险。
- 修复旧存档里非法或重复分部影响分部数量和费用的问题。
- 修复旧存档里缺少飞机分配字段时可能导致可用飞机计算异常的问题。
- 修复跨季度总结中报纸和财报时间口径不清的问题。
- 修复移动端相近城市触控热区重叠时可能选错城市的问题。
- 修复选择总部或分部时新手引导可能遮挡操作的问题。
