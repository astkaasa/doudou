# Changelog

## v0.7 - 2026-06-30

### 新内容

- 开始界面新增版本日志入口，可在开局前查看当前更新内容。

### 规则调整

- “辣豆基金”从每季度当前现金 10% 下调为 2.5%。

## v0.6 - 2026-06-30

### 体验优化

- 地图航线改为按球面大圆路径绘制，远距离航线显示更接近真实飞行路径。

## v0.5 - 2026-06-29

### 新内容

- 新增 NASDOU 证券市场，可按时代交易金融、能源、文化、旅游和科技板块股票。
- 报纸现在会显示 NASDOU 综合指数，新闻和油价会影响相关板块涨跌。
- 持仓股票会在 Q4 结算分红，并在财报中显示投资收益和持仓市值。

### 体验优化

- 底部状态栏新增 NASDOU 入口，可随时查看行情、持仓、浮盈和交易按钮。
- 旧存档会自动补齐证券市场状态，避免读档后缺失新功能字段。

## v0.4 - 2026-06-28

### 新内容

- 新增“里程碑”奖杯陈列室，会记录航线、经济、机队、分部、财富和生涯目标。
- 新增随时代变化并按季度增长的城市市场数据，航线需求会跟随人口、商务和旅游热度变化。
- 同步 upstream 城市数据，新增河内、卡拉奇、阿姆斯特丹、旧金山、拉各斯等城市。
- 灾害新闻扩展到东亚、东南亚、南亚、中东、加勒比等细分区域。

### 规则调整

- 开设分部现在需要 1 个季度施工，完工后才可作为航线起飞基地。
- 航线收入改为按距离分层的班次模型，短途高频、远程低频；票价建议、开航成本、航线落地费和维护成本同步 upstream 最新参数。
- 各时代初始现金同步 upstream 最新参数，2000 年代开局资金更充裕。
- “辣豆基金”改为按当前现金 10% 固定结算，不再随机给出 1-100M。
- 中东、南亚等城市归入亚洲下的细分区域，灾害和需求事件按细分区域匹配。
- 灾害事件现在通过事件修正器让受影响区域航线临时停飞。
- 银行贷款弹窗不再提供一次性红包资金入口。

### 体验优化

- HUD 和重读按钮改为“上季利润 / 上季报纸 / 上季财报”，季度结算口径更清楚。
- 财报航线明细改为按总部/分部汇总基地收益，更容易看出各基地表现。
- 基地收益现在可以展开查看单条航线明细，便于定位赚钱或亏损航线。
- 城市和航线选择增加市场热度、趋势以及人口/商务/旅游提示，开线前更容易判断市场潜力。
- 移动端 HUD 压缩里程碑、存档和读档等低优先级文字，保留核心经营指标可读。
- 城市选择列表优先按细分区域分组，新增城市后更容易查找。

### Bug 修复

- 修正“辣豆豆”特质说明，使其与当前资金 10% 的辣豆基金规则一致。
- 修正负现金时“辣豆基金”可能变成额外扣款的问题。
- 修正航线频率收入计算，避免航线自身 frequency 与距离班次重复乘算。
- 修复移动端奖杯陈列室可能横向溢出的问题。

## v0.3 - 2026-06-27

### 新内容

- 新增城市：檀香山。
- 新增开局“金豆特质”：随机获得辣豆基金、维护费降低或油耗降低等公司特性。
- 新增航线管理表格：支持排序、分页、单独调价、停飞/复飞、换飞机和关闭确认。

### 体验优化

- 顶部 HUD 增加公司名和特质徽章，经营状态更明确。
- 金豆特质抽取会固定本局待选顺序，避免重复打开时重新随机。
- 移动端航线管理改为卡片式列表，调价、换机、停飞和关闭操作更适合触控。

### 规则调整

- 开通航线现在需要支付一次性开通费用，长距离和高等级城市航线费用更高。
- 停飞航线不产生客座率、收入和成本，但执飞飞机仍被占用；停飞/复飞后需要推进一回合才能反向操作。
- 调价和更换执飞机型会保留上季度指标，航线管理中会标记变化，方便观察下一季度表现。

### Bug 修复

- 修复旧存档航线字段不完整时可能导致航线管理显示异常的问题。

## v0.2 - 2026-06-25

### 新内容

- 新增“银行贷款”：可以根据机队和航线规模申请贷款，扩张更快，但每季度需要支付利息。
- 新增“辣豆红包”：一次性资金援助，适合开局或现金紧张时救急。
- 新增“分部/基地”玩法：除了总部，还可以在其他城市开设分部；航线只能从总部或分部起飞。
- 新增“飞机租赁”：可以用更低的前期成本扩充机队，但租赁数量受自有飞机数量限制。
- 新增“飞机交付”：购买飞机后不再立即入队，需要等待交付；租赁飞机仍可立即使用。
- 新增机型服役和退役年份：不同年代可购买的飞机会随时间变化。
- 新增航空新闻：新机型服役、经典机型退役会出现在报纸中。
- 新增季度总结：推进回合后会同时看到新季度报纸和上一季度财报。
- 新增财报/报纸重读：可以随时回看当季报纸和上次财报；读档后也能继续重读上次财报。
- 新增飞机交付通知：飞机完成交付后会有提醒。
- 新增设置界面：可以切换地图样式和是否显示国界。

### 体验优化

- 游戏名称和默认公司名更新为“空豆霸业 / 豆豆航空”。
- 城市和飞机数据同步 upstream 的新版本。
- 购买飞机界面按厂商分组，并默认展开当前资金可负担的机型组。
- 顶部 HUD 优化显示，避免在窄窗口下文字被挤成逐字换行。
- 机队数量显示统一：HUD、机队面板和财报现在口径一致。
- 开局选择总部时，右侧城市列表现在可以直接点击选择城市。
- 地图保留当前工程版的矢量底图、缩放、拖动、横向循环和城市热区体验。
- 移动端地图支持单指拖动、双指捏合缩放，并优化相近城市的触控选择。
- 优化移动端地图拖动和缩放时的流畅度。
- 移动端底部操作收敛为“推进回合”，减少重复入口，主要经营操作回到面板中完成。
- 移动端推进按钮复用主界面的“推进回合 ▶”，保持文案和样式一致。
- 移动端帮助按钮固定在底部状态栏右侧，操作提示改为单独一行显示。
- 修正选择总部或分部时帮助按钮的位置，保持固定在底部状态栏最右侧。
- 面板滚动条更醒目，移动端浏览下方面板时更容易发现可滚动内容。
- 新手引导支持“知道了 / 不再提示”，并提供“？”帮助入口，可随时查看玩法说明或重新开启提示。
- 新手引导提示更完整，会提示开航线、买飞机、推进季度、开设分部等关键操作。
- 财报会显示贷款利息、贷款余额、购买/租赁机队数量等信息。
- 关闭分部前会提示受影响航线和飞机。

### 规则调整

- 航线只能从总部或分部起飞。
- 关闭分部会关闭从该分部起飞的航线，相关飞机回到空闲状态。
- 租赁飞机不能当作自有飞机出售，只能退租。
- 贷款额度与航线和机队规模相关。
- 贷款利息会进入季度成本。
- 飞机购买会受当前年份和服役期限制。
- 票价必须是有效的正整数。

### Bug 修复

- 修复购买飞机后 HUD 机队数量和机队面板/财报不一致的问题。
- 修复重读财报时可能混用旧利润和当前现金/机队状态的问题。
- 修复读档后无法可靠重读上次财报的问题。
- 修复开局选择总部时右侧城市列表无法点击的问题。
- 修复还款金额为 0 时仍可能显示成功的问题。
- 修复租赁飞机可能被错误出售的风险。
- 修复旧存档里非法或重复分部影响分部数量和费用的问题。
- 修复旧存档里缺少飞机分配字段时可能导致可用飞机计算异常的问题。
- 修复跨季度总结中报纸和财报时间口径不清的问题。
- 修复移动端相近城市触控热区重叠时可能选错城市的问题。
- 修复选择总部或分部时新手引导可能遮挡操作的问题。

## v0.1 - 2026-06-24

### 首个本地版本

- 基于 upstream 原始版本整理出本地可运行版本。
- 普通玩家可以直接双击 `dist/standalone.html` 开始游戏。
- 保留原版航空经营核心玩法：选择时代、选择总部、购买飞机、开通航线、调整票价、推进季度。
- 加入本地存档和读档。
- 加入季度报纸事件和财报。
- 加入 AI 竞争航线。
- 加入深色管理面板风格界面。
- 加入世界地图、城市点、航线显示和城市选择。

### 地图更新

- 地图改为本地矢量底图，不再依赖运行时下载。
- 支持城市点击、hover 城市名、滚轮缩放、拖动和横向循环滚动。
- 改善手机触控热区，城市点更容易点击。

### 稳定性

- 补充了经济、航线、事件、存档和回合推进等基础测试。
- 构建流程可以生成普通静态版本和单文件 standalone 版本。
